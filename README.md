# Personal Blog

My personal blog built with [Astro](https://astro.build), Vue, and Tailwind CSS. Deployed on Vercel.

## Getting started

```sh
pnpm install
pnpm dev
```

## Commands

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `pnpm dev`     | Start local dev server at `localhost:4321`  |
| `pnpm build`   | Build for production to `./dist/`           |
| `pnpm preview` | Preview the production build locally        |
